# 3i-entry-exit

シューティングお化け屋敷のタイムアタックランキングシステムのWeb側です。

## Requirement

- yarn 1.22.10

## Installation

1. clone repository
```sh
git clone git@github.com:Futaba-Kosuke/3i-entry-exit-client.git
cd 3i-entry-exit-client
```

2. install modules
```sh
yarn install
```

## Usage

```bash
# Start the server
yarn dev
```

## Used
- Vue.js
